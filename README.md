# 🚀 AI LinkedIn Content Automation (n8n)

This project helps automate the process of creating and managing daily LinkedIn content using AI and n8n.

## 🔥 Features
- Generates fresh and unique AI-related posts every day  
- Uses simple, easy-to-read language suitable for LinkedIn  
- Sends generated content directly to Telegram for quick review  
- Fully automated workflow with minimal manual effort  

## ⚙️ Workflow
Schedule Trigger → OpenAI → Set → Telegram

## 📦 Setup
1. Import the `workflow.json` file into n8n  
2. Add your OpenAI API key  
3. Configure your Telegram Bot Token and Chat ID  
4. Activate the workflow  

## 💡 Use Cases
This project is useful for:
- Building a consistent personal brand on LinkedIn  
- Sharing insights about AI tools and trends  
- Automating daily content creation  

## 🚀 Outcome
This workflow reduces the time spent on content creation and helps you stay consistent with high-quality LinkedIn posts.
